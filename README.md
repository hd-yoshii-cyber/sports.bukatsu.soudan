# スポーツ現場の実務メンターマッチング

スポーツ・部活動の現場で起きる実務的な悩みを、経験ある実務メンターに相談できるマッチングプラットフォーム。

## 新ビジネスモデル

### サービスの位置づけ
- 対象：指導者、保護者、チーム運営者、クラブ関係者、18歳以上の競技者
- 18歳未満は保護者経由申込を原則
- サッカーを含むが特定競技に限定しない（野球、バスケ、バレー、陸上、テニス、卓球、水泳、武道など）

### 登録メンター
- ベテラン指導者、スポーツトレーナー、クラブ運営経験者、メンタルコーチ、チームビルディング支援者

### 対象外
- 法律相談、税務相談、医療行為、診断、治療、心理療法
- 雇用あっせん・採用仲介
- 弁護士、税理士、医師等の資格業務

## 相談方式と4プラン料金（完全固定制）

メンターが金額を指定する仕組みは廃止。完全固定制。

| プラン | 方式 | 料金（税込） | システム利用料 | メンター送金目安 |
|--------|------|-------------|---------------|----------------|
| A | メール相談（400字/1往復） | 5,000円 | 1,000円+決済手数料 | 約3,700円 |
| B | メール相談（400字/1往復） | 10,000円 | 1,000円+決済手数料 | 約8,700円 |
| C | 音声通話 30分 | 12,000円 | 3,000円+決済手数料 | 約8,600円 |
| D | 音声通話 60分 | 20,000円 | 3,000円+決済手数料 | 約16,300円 |

- 決済手数料：約3.6%
- 表示料金以外の追加請求なし
- 相談料にはシステム利用料含む

## マッチング条件カテゴリ

相談者が投稿時に選択 / メンターが登録時に選択（同一カテゴリ）

### 指導カテゴリ
- 小学生 / 中学生 / 高校生 / 大学生 / 社会人チーム

### 実績カテゴリ
- 全国大会出場 / 都道府県上位入賞 / 10年以上 / 20年以上の指導歴

### 所属・環境カテゴリ
- 公立中学 / 公立高校 / 私立中高 / 大学 / クラブチーム / Jリーグトップ / Jアカデミー / Jスクール / 実業団・社会人

### 専門分野カテゴリ
- GK / フィジカル / メンタル / チームビルディング / 保護者対応 / クラブ運営

## 決済構造

1. 相談投稿（無料）
2. 事務局がメンター候補を紹介
3. 相談者がメンターを選択
4. プラットフォーム上で決済
5. 相談実施
6. 相談料からシステム利用料・決済手数料を控除してメンターへ送金
